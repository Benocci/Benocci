Computer Science graduate focused on efficient indexing and compression techniques for high-dimensional vector representations produced by machine learning models.

My main interests lie in **Approximate Nearest Neighbor (ANN) search**, **compressed indexing**, and **scalable information retrieval systems** for modern AI applications.

I enjoy working close to the system level, building **efficient data structures and high-performance implementations** for large-scale data processing.

---

## Research Interests

- Approximate Nearest Neighbor (ANN) search
- Vector similarity search
- High-dimensional indexing structures
- Graph-based search algorithms (HNSW)
- Compression techniques for large indexes
- Efficient data structures for information retrieval
